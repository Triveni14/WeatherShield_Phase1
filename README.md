WeatherShield AI

* Project Overview
WeatherShield AI is a simple frontend application that simulates income protection for workers based on weather conditions.

* Features
. Login and Signup pages
. Dashboard with worker details
. Plan selection (₹30 / ₹50 / ₹80 per week)
. Weather simulation (Rain, Heat, Normal)
. Automatic payout display based on weather

* How it works
. User logs in and selects a protection plan
. Weather conditions are simulated using buttons
. Based on condition:
  - Rain → ₹500 credited
  - Heat → ₹300 credited
  - Normal → No payout

* Tech Stack
 . React (Vite)
 . JavaScript
 . CSS

* NOTE
   This a frontend-only simulation. Backend and real weather API integration will be added in future phases.
