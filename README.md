*** WeatherShield AI

**Project Overview
WeatherShield AI is a simple frontend application that simulates income protection for workers based on weather conditions.

** Features
- Login and Signup pages
- Dashboard with worker details
- Plan selection (₹30 / ₹50 / ₹80 per week)
- Weather simulation (Rain, Heat, Normal)
- Automatic payout display based on weather

** How it Works
1. User logs in and selects a protection plan
2. Weather conditions are simulated using buttons
3. Based on condition:
   - Rain → ₹500 credited
   - Heat → ₹300 credited
   - Normal → No payout

** Tech Stack
- React (Vite)  
- JavaScript  
- CSS  

** NOTE
This is a **frontend-only simulation**. Backend and real weather API integration will be added in future phases.


** Future Plans
- Backend Integration: Add a server to store user accounts and plan data  
- Real Weather API: Integrate live weather data to simulate payouts based on actual conditions  
- Payment Gateway: Enable users to subscribe to plans using real payments  
- Analytics Dashboard: Show historical payouts and user statistics
